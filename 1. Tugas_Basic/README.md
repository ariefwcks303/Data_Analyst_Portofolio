# Data Analytics Project: Customer Purchase Behavior Analysis

Proyek ini bertujuan untuk menganalisis perilaku pembelian pelanggan menggunakan **Microsoft Excel**. Melalui pengolahan data mentah dan penggunaan **Pivot Table**, proyek ini menghasilkan *insight* mendalam serta rekomendasi strategis untuk meningkatkan performa bisnis.

### Part 1: Data Cleaning & Preprocessing
Sebelum melakukan analisis, saya melakukan pembersihan data untuk memastikan validitas:
* **Handling Duplicates:** Memeriksa dan menghapus baris data ganda agar tidak terjadi *double-counting*.
* **Missing Value Treatment:** Mengisi nilai yang kosong dengan angka `0` untuk menjaga konsistensi kalkulasi.
* **Text Formatting:** Merapikan penulisan yang tidak beraturan menjadi format kapital di awal kata (*Proper Case*).
* **Data Type Conversion:** Mengonversi format data pada kolom `Age` dan `Purchase` menjadi numerik agar dapat diolah dalam operasi matematika.

### Part 2 & 3: Data Exploration & Visualization
Menggunakan **Pivot Table** dan **Excel Charts**, saya mengeksplorasi beberapa metrik utama:
* Analisis total pembelian berdasarkan gender dan tingkat pendapatan.
* Identifikasi kategori produk dengan performa penjualan tertinggi (*Best-Seller*).
* Pemetaan kota dengan pelanggan paling aktif berdasarkan frekuensi belanja.
* Analisis korelasi antara penggunaan diskon dengan frekuensi pembelian.
* Evaluasi dampak pengaruh media sosial terhadap besarnya nilai transaksi.

## ğŸ’¡ Strategic Recommendations

Berdasarkan hasil analisis, berikut adalah rekomendasi strategi yang diusulkan:

### 1. Strategi Pemasaran Berbasis Demografi
* **Targeting:** Memprioritaskan penargetan pada wanita berpendapatan tinggi dan melakukan personalisasi pengalaman belanja di semua segmen wanita.
* **Ekspansi:** Mengidentifikasi peluang pertumbuhan pada segmen pria dan mengoptimasi portofolio produk berdasarkan preferensi gender.
* **Seasonal Campaign:** Menjalankan kampanye khusus seperti *"Happy Weekend with Gardening"* atau diskon musiman untuk target usia 35+ pada kategori terkait.

### 2. Optimalisasi Produk (Jewelry & Accessories)
* Memaksimalkan potensi kategori **Jewelry & Accessories** sebagai kontributor utama.
* Melakukan analisis mendalam untuk meningkatkan kategori dengan penjualan menengah/rendah serta mengoptimalkan tampilan produk (*visual merchandising*).

### 3. Pengembangan Berbasis Wilayah (Geo-Targeting)
* **Bandung (Kota Teraktif):** Fokus pada program apresiasi, retensi, serta pengambilan umpan balik mendalam (*feedback*) untuk uji coba produk baru.
* **Kota Lain:** Mengidentifikasi hambatan aktivitas dan menjalankan kampanye pemasaran bertarget untuk meningkatkan keterlibatan pelanggan.

### 4. Manajemen Diskon & Loyalitas
* **High-Frequency Shoppers:** Mendorong pembelian lebih lanjut melalui diskon strategis dan program loyalitas berbasis poin.
* **Analisis Non-Diskon:** Memberikan nilai tambah non-diskon bagi pelanggan yang tetap membeli tanpa potongan harga.
* **Testing:** Melakukan *A/B testing* pada berbagai jenis insentif dan memantau *churn rate*.

### 5. Strategi Media Sosial & Influencer
* **Influencer Marketing:** Melibatkan influencer dan program afiliasi untuk menciptakan konten yang mudah dibagikan.
* **Segmentasi Konten:** Memfokuskan edukasi pada segmen "Low/None" sosial media agar terkonversi menjadi pembeli aktif, serta mengukur ROI dari setiap kampanye digital.
